== Mini Blue Box Boy ==
By Mr. Wonko
A game made in 48h for Ludum Dare 22 (December 2011)
Updated to latest SFML 2 version in December 2012

Requires the Microsoft Visual C++ 2010 Redistributable Package (x86)

You are Blue Box Boy and you're all alone. So you run and jump around, hoping to find somebody. But deep inside you already know: You won't find anybody. But maybe you'll win!

Source code available on https://github.com/mrwonko/Ludum-Dare-22

Controls:
A - Left
D - Right
W, Space - Jump

Cheats:
F1 - Enable Level Editor
F2 - Next Level
F3 - Previous Level
P - Physics Debug Overlay
